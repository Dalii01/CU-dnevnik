# Электронный школьный дневник

Проект представляет собой веб-приложение для управления учебным процессом — электронный дневник с поддержкой различных ролей (ученики, родители, учителя и администраторы). Система позволяет вести учет оценок, просматривать расписание и получать уведомления об успеваемости.

## Основной функционал

* **Персонализированные профили**: удобное отображение данных каждого пользователя.
* **Расширенная статистика успеваемости**: 
  * Расчет среднего и медианного балла учащегося.
  * Визуализация распределения оценок с помощью графиков-баров.
  * Учителя могут видеть средний балл каждого ученика прямо на главной странице.
* **Безопасность и контроль доступа**:
  * Строгие политики создания логинов и паролей для предотвращения взломов.
  * **Родительский контроль**: родители имеют доступ к дневникам *только* прикрепленных к ним детей (настраивается администратором).
  * **Управление пользователями**: возможность деактивации (блокировки) и активации аккаунтов администратором.
* **Удобное расписание**: наличие фильтров для быстрого поиска занятий по предмету, дню недели или времени.
* **Telegram-уведомления**: автоматическая отправка сообщений о новых полученных оценках и комментариях учителей в Telegram (через интеграцию с Telegram Bot API).
* **Расширяемость**: архитектура проекта позволяет легко добавлять новые фичи (например, аналитику, геймификацию или автоматизацию работы преподавателей).

## Стек и архитектура

* Проект написан на **Python** (интеграция с Telegram Bot API).
* Используется база данных для хранения профилей, расписания и оценок.

## Структура проекта

* `src/` — исходный код приложения.
* `instance/` — конфигурация и данные локального экземпляра.
