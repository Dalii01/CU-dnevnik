Обновить Telegram ID

Укажите ваш Telegram ID для получения уведомлений о новых оценках

Как получить Telegram ID?

  1. Откройте Telegram и найдите бота @userinfobot
  2. Отправьте команду /start
  3. Бот покажет ваш ID (число) или username (@username)
  4. Скопируйте и вставьте сюда

Форматы ID

  • Числовой ID: например, 123456789
  • Username: например, @myusername
  • Пустое поле: отключить уведомления
